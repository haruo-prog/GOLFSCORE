# GOLFSCORE

Android向けのゴルフスコア管理アプリです。

## Version

- App version: 1.8.0
- V2 workflow redesign
- versionCode: 19
- compileSdk: 36
- targetSdk: 36
- minSdk: 26

## Main features

- Home, history, analysis, and settings screens
- Round input screen focused on one-hole input
- Player1 detailed analysis
- Player2 to Player4 score-only input
- PAR-based score buttons
- Big score cycle button from 9 to 15
- Club set settings
- Tee shot club tracking
- Course template save and load
- Course, hole, and club analysis
- Post-round advice
- Scorecard PDF export
- Remembered PDF destination
- Backup and restore through Android file picker
- Club photo WebP compression
- NKTS launcher icon
- Autosave
